You are upgrading the Ziba Rider App UI to a mature, professional, production-ready design.

--------------------------------
STEP 1: DEFINE ZIBA DESIGN SYSTEM
--------------------------------

Create a centralized design system file and enforce it across Rider screens.

COLORS:
- PrimaryColor: use Ziba brand primary color ONLY for:
  - Primary CTA buttons
  - Active ride status
  - Progress indicators
- BackgroundColor: light neutral (cards sit on this)
- CardColor: white or near-white
- TextPrimary: dark neutral (headings)
- TextSecondary: muted gray (supporting text)
- SuccessColor: soft green (ARRIVED, COMPLETED)
- DangerColor: muted red (Cancel only)

RULES:
- Never use bright colors for body text
- Never mix multiple accent colors on one screen
- No red/orange for non-critical states

TYPOGRAPHY:
- Headline (large, semibold): status titles and screen headers
- Body (regular): descriptions and instructions
- Caption (small): metadata like ETA, vehicle info

No decorative fonts.
No excessive bolding.
No inconsistent font sizes.
